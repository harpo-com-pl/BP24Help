New email message:
- Previous field: Space + dots 23, thumb key Previous (far left)
- Next field: Space + dots 56, thumb key Next (far right)
- Subject field: Space + dots 2348 (Space + Enter + s)
- CC field: Space + dots 148 (Space + Enter + c)
- Recipient field: Space + dots 12358 (Space + Enter + r)
- BCC field: Space + dots 1258 (Space + Enter + h)
- Message body: Space + dots 158 (Space + Enter + e)
- Contacts list: Space + dots 124 (Space + f)

Reading an email message:
- Open the message: Dot 8 (Enter), any cursor invocation button
- Navigate through the text: Thumb keys
- Attachment menu: Space + dots 18 (Space + Enter + a)
- Exit the message: Space + dots 14 (Space + c)
